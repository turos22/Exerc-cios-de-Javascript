# ⚡ Exercícios de JavaScript

Um repositório dedicado ao estudo e prática de JavaScript fundamentais. O objetivo deste projeto é consolidar conceitos de lógica de programação, manipulação do DOM e interatividade no Front-End através de pequenos desafios práticos.

## 🛠️ Estrutura do Projeto

Os exercícios foram unificados em uma interface simples para facilitar o teste das funções:

* **index.html:** Estrutura de visualização e interface dos exercícios.
* **estilo.css:** Estilização básica e organização visual dos elementos.
* **funcoes.js:** Onde a mágica acontece. Contém as resoluções dos exercícios e toda a lógica de programação desenvolvida.
* **imagens/:** Diretório reservado para mídias utilizadas na interface.
* **screenshots/:** Pasta com o registro visual da interface do projeto executando.

---

## 🚀 Conceitos Praticados

Durante o desenvolvimento destes exercícios, foquei em aplicar:
* Estruturas condicionais (`if`, `else`) e de repetição.
* Criação, declaração e chamada de funções adaptáveis.
* Manipulação de eventos no HTML (como cliques de botões).
* Captura e exibição de dados dinâmicos na tela para o usuário.

---

## 💻 Como Executar os Exercícios

Se quiser testar os códigos localmente:

1. Clone o repositório na sua máquina:
   git clone https://github.com/turos22/Exerc-cios-de-Javascript.git

2. Acesse a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador para interagir com a página de exercícios.
